📝 7 JavaScript Assignments on Date & Time (اسائنمنٹس)

    1. Assignment 1:

  • 🛠 اپنی سالگرہ کی تاریخ کا دن معلوم کرنے کے لیے JavaScript کا استعمال کریں۔
  • 🔹 Write JavaScript to find the day of the week for your birthday.
  • 💡 Hint: Use getDay() (0 = Sunday, 1 = Monday, etc.)

    • 2. Assignment 2:

  • 🛠 پاکستان اور نیویارک کے موجودہ وقت کا موازنہ کرنے کے لیے JavaScript کوڈ لکھیں۔
  • 🔹 Write JavaScript to compare the current time in Pakistan and New York.
  • 💡 Hint: Use Intl.DateTimeFormat() for different time zones.

    • 3. Assignment 3:

  • 🛠 12 گھنٹے کے فارمیٹ کو 24 گھنٹے کے فارمیٹ میں تبدیل کرنے کے لیے ایک فنکشن لکھیں۔
  • 🔹 Write a function to convert 12-hour format time to 24-hour format.
  • 💡 Hint: Use split(":") and parseInt().

    • 4. Assignment 4:

  • 🛠 مندرجہ ذیل تاریخوں میں غلطیوں کو درست کریں اور درست تاریخ حاصل کریں:
  • 🔹 Correct the following incorrect dates using JavaScript:
  • "31/02/2023"
  • "12-13-2024"
  • "25/12/2025"
  • 💡 Hint: Check if the date is valid using the Date object.

    • 5. Assignment 5:

  • 🛠 دنیا کے پانچ مختلف ممالک کے موجودہ وقت کا موازنہ کریں۔
  • 🔹 Compare the current time in five different countries.
  • 💡 Hint: Use Intl.DateTimeFormat() with different locales.

    • 6. Assignment 6:

  • 🛠 "ٹائم زون" (Time Zones) کیا ہوتے ہیں؟ جاوا اسکرپٹ میں کم از کم تین مختلف ٹائم زونز کے مطابق وقت دکھائیں۔
  • 🔹 Explain time zones and display the time in at least three different zones using JavaScript.
  • 💡 Hint: Use toLocaleString() with different time zones.

    • 7. Assignment 7:

  • 🛠 ایک گاڑی 2:30 PM پر کراچی سے روانہ ہوتی ہے اور 5 گھنٹے 15 منٹ بعد لاہور پہنچتی ہے، تو وہ کس وقت لاہور پہنچے گی؟
  • 🔹 A car leaves Karachi at 2:30 PM and arrives in Lahore after 5 hours and 15 minutes. Write JavaScript to calculate the arrival time.
  • 💡 Hint: Use setHours() and setMinutes().

✅ یہ اسائنمنٹس مکمل کرنے کے بعد، آپ جاوا اسکرپٹ میں تاریخ اور وقت کے بارے میں مہارت حاصل کر لیں گے۔
✅ By completing these assignments, you’ll master JavaScript’s Date & Time handling.
